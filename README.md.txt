# Social Media Viral Boost System

This repository contains the backedn design of a social media platform
where posts and videos are boosted based on engagement metrics.

Core Modules
1 Upload Photo / Video
2 Feed System
3 Like / Comment / Share
4 Viral Boost Algorithm

The system calculates a viral score using:
Likes 
Comments
shares
Watch Time

Posts with higher engagement are promoted to the trending feed.

Project Structure
docs-> BRD and PRD documentation
models-> Database schemas
controllers-> Business logic
routes-> API endpoints
services-> Viral ranking algorithm
